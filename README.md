# 💰 KPS Tracking - Personal Finance Management System

Aplikasi manajemen keuangan pribadi berbasis desktop dengan Java Swing dan MySQL.

---

## 📋 **Fitur**
- Login & Register
- CRUD Category (Pemasukan/Pengeluaran)
- CRUD Transaksi
- Laporan & Pencarian ( otw )
- Session Management (Guest/Login)

---

## 🛠️ **Tech Stack**
- Java 17, Swing, MySQL, JDBC, NetBeans


## 🚀 **Cara Jalankan**
1. Clone repo
2. Import database
3. Setting Database.java
4. Clean & Build
5. Run

---

## ✅ **Kelebihan**
- MVC
- CRUD lengkap
- Session & autentikasi
- Popup konfirmasi
- Relasi database

---

## ❌ **Kekurangan**
- Belum ada grafik
- Belum export PDF/Excel
- Password belum hash
- UI standar
- Desktop only
